ENGLISH (Englisch)
----------------------------------------------------------

Installation:
1.) Make a copy of your existing "Data-1.13" folder in your JA2 root directory and rename it to "Data-WildFire6.06".
2.) Make a folder where you would like to download the files. Use TortoiseSVN "SVN Checkout..." and browse to
	https://81.169.133.124/source/ja2/branches/Wanne/1.13 Wildfire Hybrid 6.0
3.) Download all the files with TortoiseSVN to the folder.
4.) After download (update from TortoiseSVN) copy the content to your JA2 root directory. Overwrite any existing files
5.) Start new game

This is the current revision history of the NIV1.13WF6.06_hybrid_international, status: alpha

Content:
- Loading- and options screens from Wildfire 6.06 for all 3 resolutions (640x480, 800x600, 1024x768)
- ALL maps and alternative maps from Wildfire 6.06. Some of them will never be used because of the internal behaviour of Ja2 v1.13 EXE.
- Added item crate for the Drassen aiport map which is used for Bobby Ray delivery. This crate is located in the north area of the map.
- Removed alarm triggers fom drug-cartel maps near San Mona. (There were someproblems with enemy AI in those maps).
- Replaced "problematic" items in many maps with "meaningful" items.
- Replaced Ira's face with alternative face from Wildfire 6.06.
- Reworked XML files (AltSectos.xml, Samsites.xml).
- New/replaced Arulco overview map, MilitiaMap.sti, ArmyComposistion, GarrisionGroups.xml
- Checked San Mona quests and scripts for errors with v1.13, reworked and tested. Spike now works leadeship around 40 +/- 5. Kyle is still in progress.
- Replaced laptop style with laptop from Wildfire 6.06
- Some smaller changes
